# agra  

`agra` is a four player multiplayer game based on the board game "Aggravation". The front end was built with `create-react-app`. The app uses a Node Express back end.

---
### Getting Started  

To start the front end:  
`cd client`  
`npm install`  
`npm start`  

To start the server:  
`cd server`  
`npm install`  
`npm run dev`
